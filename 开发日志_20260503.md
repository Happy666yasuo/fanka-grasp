## 代码更新

### Backend 选择

`create_pick_place_simulation` 现在支持：

```text
explicit backend argument
  -> EMBODIED_SIM_BACKEND
  -> mujoco default
```

新增环境变量：

```bash
EMBODIED_SIM_BACKEND=mujoco|pybullet|isaaclab
```

显式 `backend=` 参数优先级高于环境变量，避免测试或脚本被外部环境意外改写。

## 已知遗留问题

1. MuJoCo backend 仍是 kinematic，不是完整 Franka/Panda dynamics。
2. IsaacLab backend 只是已注册占位，还没有真实训练环境。
3. Chart generator 对 comparative 和 ablation 使用部分相同图片文件名，后跑的图会覆盖同名文件。
4. 运行 Phase3 模块时仍有 `runpy` warning，原因是 package import 后再次以 `-m` 执行同模块；暂不影响结果。
5. Matplotlib cache 目录不可写时会使用 `/tmp` 临时目录；暂不影响报告生成。
